# IAM Control Subset

IAM was used as an external control benchmark for HTR evaluation.

The experiments used a 200-sample subset from the `Teklia/IAM-line` Hugging Face dataset. IAM was not used as a training dataset in this thesis; it was used to check zero-shot behavior and domain generalization.

The IAM images are not stored in this repository. They can be loaded from Hugging Face when rerunning the notebooks or scripts.
